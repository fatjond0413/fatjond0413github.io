AVD Jigsaw - Animation

@Keyframes

Keyframes is a code that allows for the steps of animation to be animated. The @keyframes rule determines and controls what happens over the course of the animation by using percentages from 0% to 100%. An example of this is:


    @keyframes color {
      0% {
        background-color: blue;
      }
      100% {
        background-color: yellow;
      }
    }

Position

The property position basically affects where the element being animated will be located at.


     @keyframes rainbow {
     0% {
       background-color: blue;
       top: 0px;
       position: fixed; 
      }
     }

Opacity

The property opacity changes how transparent the element is so if the opacity is 0.1 it is almost see through but if the opacity is 1 it will not be see through.


    @keyframes fade {
      50% {
        left: 60%;
        opacity: 0.1;
      }
    }

animation-name

The animation-name sets the name of the animation as the name implies. An example for this element in code is: animation-name: star-1;

animation-duration

The property animation-duration sets how long the animation runs for. How it works is: animation-duration: 10s;

animation-iteration-count

The animation-iteration-count property allows you to run the animation as many times as you want. You can set the value to any number or infinite which will make the animation run forever.


    #count {
      font-weight: bold;
      padding: 5px;
      animation-name: color;
      animation-duration: 4s;
      animation-iteration-count: infinite;
    }

    @keyframes color {
      0% {
        background-color: darkred;
      }
      50% {
        background-color: green;
      }
      100% {
        background-color: blue;
      }
    }

animation-fill-mode

The style applied to an element when the animation has ended is specified by the animation-fill-mode property and the value must be set to forwards in order to keep the animation running forever.


      #f {
        border-radius: 10px;
        padding: 5px 10px 8px 10px;
      }
      #f:hover {
        animation-name: background-color;
        animation-duration: 1s;
        animation-fill-mode: forwards;
      }
      @keyframes background-color {
        100% {
          background-color: #aa8555;
        }
      }

animation-timing-function

This property controls the speed of the change in an animated element over the duration of the animation. The different values of the animation-timing-function are:

  • animation-timing-function: ease; - from slow to fast to slow
  • animation-timing-function: ease-in; - from slow to fast
  • animation-timing-function: ease-out; - from fast to slow
  • animation-timing-function: linear; - constant speed

Cubic bezier curves, used with the cubic-bezier function, provide a better control over how the animation runs. They run on a 1 by 1 coordinate system, the X-axis is the duration of the animation and the Y-axis is the change in the animation. They can be used instead of the other keywords for the animation-timing-function. For example:

  • animation-timing-function: cubic-bezier(0.0, 0.0, 1.0, 1.0); = linear
  • animation-timing-function: cubic-bezier(0.42, 0, 1.0, 1.0); = ease-in
  • animation-timing-function: cubic-bezier(0, 0, 0.58, 1); = ease-out

Transform

The transform property allows you to change the size of an element. Here's an example of how to use it: transform: rotate(90deg);.
